 -- GAMEPLAY --
 
 - Clicking on the GUI starts the game
 - Black Player always moves first
 - Game ends when a player runs out of pieces
 - Game Over screen comes up
 - Can initiate another game by clicking the screen again
 - Exit the game by closing the window
 
 -- INTERFACE --
 
 - Click on a checker to highlight it's possible moves
 - Click on any highlighted region to move the checker
 - To cancel your move, click outside the tiles that you can move to
 
 -- RUNNING THE GAME --
 
 - Game runs as soon as you start the program
